### Step 1: Navigate to Repository
Go to the **Repository** section of your system (left sidebar or navigation menu)

### Step 2: Find a Document with QPRO Analysis
Look for documents that have been uploaded with QPRO analysis performed

### Step 3: Click Preview
Click the preview button/document title to view the document

## When is QPRO Analysis Available?

The QPRO analysis section appears when:
- A document has been analyzed with QPRO
- The analysis has been created (even if still in DRAFT status)
- You have permission to view it

The section **will not appear** if:
- No QPRO analysis exists for the document
- You don't have access to view the analysis
- The document hasn't been uploaded to QPRO yet

## Permissions

### Who Can View Analysis?
- **Admins**: Can view all analyses
- **Faculty**: Can view all analyses
- **Other Users**: Can only view their own analyses

### Status Indicators
- **DRAFT**: Analysis is being reviewed, not yet committed
- **APPROVED**: Analysis has been approved and committed to dashboard
